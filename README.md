#  Support Ticket System

A full-stack Support Ticket Management System built using Django, React, PostgreSQL, and OpenAI.  
The system allows users to create support tickets and automatically classifies them using an LLM.

---
# Features

- Create support tickets
- Automatic ticket classification using OpenAI (GPT-4o-mini)
- View all tickets
- Ticket filtering and search
- Ticket statistics aggregation
- RESTful API with Django REST Framework
- Fully Dockerized setup
- Clean frontend-backend separation
- LLM failures do not block ticket creation

---

##  Tech Stack

### Backend
- Django
- Django REST Framework
- PostgreSQL

### Frontend
- React
- Fetch API

### DevOps
